## 比较
1. 选择排序，复杂度O(n^2)
2. 插入排序，复杂度最差O(n^2)，对近似有序的数组则是无限接近与O(n)，因为判断过程中存在提前break的情况，所以性能可能优于选择排序
3. 归并排序，复杂度O(nlogn)，在数据量小的时候使用插入排序,因为存在
4. 快速排序，复杂度O(nlogn)，对近似有序的数组则退化为O(n^2)，在数据量小的时候仍然建议使用插入排序
